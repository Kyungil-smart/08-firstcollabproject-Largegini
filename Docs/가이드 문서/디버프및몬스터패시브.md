1. 플레이어 ``_reverse`` bool타입
- 플레이어가 사신에게 2번째 패턴을 맞았을때 생기는 디버프 1턴간 지속되며 이 디버프가 있을 경우 회복블록을 매칭했을 시 플레이어 본인에게 대미지가 들어오게 된다. ```Player.cs```가 `_reversr`를 들고 있으며 ```Envoy.cs```에서 2번째 패턴 공격 끝나고 대미지 계산이 끝난 후 true로 바꿔준다

2. 플레이어 ``_theEnd`` bool 타입
- 플레이어가 사신에게 3번째 패턴을 맞았을때 생기는 디패시브형이며 영구지속된다. 중첩은 되지 않지만 매턴 5의 대미지를 받으며 마찬가지로 ```Player.cs```에서 `_theEnd`를 들고 있으며 ```Envoy.cs```에서 3번째 패턴 공격이 끝나고 대미지 계산 후 true로 바꿔준다.

3. 사신 ``_soulHarvest`` int 타입
- 사신이 플레이어를 공격 할 때마다 스택이 쌓이는 스택형 버프 영구지속되며 갯수당 사신이 플레이어에게 주는 대미지추가
